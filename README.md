

# 🚀 Scalable Web App with NLB and Auto Scaling

## 📌 Project Overview
This project demonstrates how to build a highly scalable and low-latency web application using AWS services like Network Load Balancer and Auto Scaling.

---

## 🎯 Purpose
- Handle high-performance traffic
- Achieve low latency
- Ensure application availability and scalability

---

## 🧰 AWS Services Used
- Amazon EC2
- Network Load Balancer (NLB)
- Auto Scaling Group

---

## ⚙️ Key Features
- Distributes traffic using NLB
- Automatically scales EC2 instances
- High availability and fault tolerance
- Health checks for instance monitoring
